How to Get Your Own Cohere API Key:

  1. Visit the official Cohere website: cohere.com/api
  2. Sign up for a free account or log in if you already have one.
  3. Navigate to the "API Keys" or "Dashboard" section.
  4. Generate a new API key. It typically starts with `cohere_`.
  5. Copy the generated key and paste it into the API key field on the Chat page.

A personal API key ensures higher usage limits and more control over your AI interactions.
